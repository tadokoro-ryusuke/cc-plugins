# UI/UX Pro Max

Professional UI/UX design intelligence plugin for Claude Code.

## Features

Searchable databases providing:
- **57 UI Styles** - Glassmorphism, Minimalism, Brutalism, Neumorphism, etc.
- **95 Color Palettes** - Industry-specific (SaaS, E-commerce, Healthcare, etc.)
- **56 Font Pairings** - With Google Fonts imports
- **24 Chart Types** - Dashboard and analytics recommendations
- **98 UX Guidelines** - Best practices and anti-patterns
- **8 Tech Stacks** - html-tailwind, react, nextjs, vue, svelte, swiftui, react-native, flutter

## Prerequisites

- Python 3.x

## Usage

The skill activates automatically when you request UI/UX work (design, build, create, implement, review, fix, improve).

Example requests:
- "Create a SaaS landing page with glassmorphism style"
- "Design a dashboard for fintech app"
- "Build an e-commerce product card with dark mode"
- "Implement a minimal portfolio website"

## Search Domains

| Domain | Description |
|--------|-------------|
| `product` | Product type recommendations (SaaS, e-commerce, portfolio) |
| `style` | UI styles, colors, effects (glassmorphism, minimalism) |
| `typography` | Font pairings with Google Fonts imports |
| `color` | Color palettes by industry/product type |
| `landing` | Landing page structures and CTA strategies |
| `chart` | Chart types and library recommendations |
| `ux` | Best practices, accessibility guidelines |
| `prompt` | AI prompts and CSS keywords |

## Supported Stacks

- `html-tailwind` (default)
- `react`
- `nextjs`
- `vue`
- `svelte`
- `swiftui`
- `react-native`
- `flutter`
